Yellowstone Trails

Trail is instance of Path. Itinerary travels by Path. Path starts from and ends at Points of Interest.

Upper Geyser Basin Boardwalk

Location: Old Faithful area

Paved boardwalk accessing thermal features

  • Path starts from: Old Faithful Visitor Center
  • Path ends at: Morning Glory Pool
  • Distance: 1.3 miles loop
  • Surface: Boardwalk and paved trail
  • Difficulty: Easy

Mount Washburn Trail

Location: Central Yellowstone

Summit trail with panoramic views

  • Path starts from: Dunraven Pass Trailhead
  • Path ends at: Mount Washburn Summit
  • Distance: 6.2 miles round trip
  • Elevation gain: 1,384 feet
  • Difficulty: Moderate

Uncle Tom's Trail

Location: Grand Canyon of Yellowstone

Steep trail to Lower Falls viewpoint

  • Path starts from: Uncle Tom's Trailhead
  • Path ends at: Lower Falls Platform
  • Distance: 0.7 miles round trip
  • Features: 328 metal steps
  • Difficulty: Moderate (steep descent)

Fairy Falls Trail

Location: Midway Geyser Basin area

Access to waterfall and Grand Prismatic overlook

  • Path starts from: Fairy Falls Trailhead
  • Path ends at: Fairy Falls
  • Distance: 5.4 miles round trip
  • Side trip: Grand Prismatic overlook
  • Difficulty: Easy

Shoshone Lake Trail

Location: Backcountry southwestern Yellowstone

Wilderness lake access trail

  • Path starts from: DeLacy Creek Trailhead
  • Path ends at: Shoshone Lake
  • Distance: 3 miles one way
  • Backcountry camping: Permits required
  • Difficulty: Easy
