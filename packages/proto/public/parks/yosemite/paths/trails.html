Yosemite Trails

Trail is instance of Path. Itinerary travels by Path. Path starts from and ends at Points of Interest.

Mist Trail

Location: Yosemite Valley to Nevada Fall

Famous granite stairway trail with waterfall mist

  • Path starts from: Happy Isles Trailhead
  • Path ends at: Vernal Fall (2.7 mi) or Nevada Fall (4.1 mi)
  • Features: 600+ granite steps, waterfall spray
  • Difficulty: Moderate to strenuous
  • Season: Best spring through fall

Valley Floor Trails

Location: Throughout Yosemite Valley

Network of paved and dirt paths connecting valley attractions

  • Path starts from: Multiple valley trailheads
  • Path ends at: Valley View, Mirror Lake
  • Distance: 1-5 miles depending on route
  • Surface: Paved and natural paths
  • Difficulty: Easy

John Muir Trail

Location: Alternative route to Nevada Fall

Less steep route with switchbacks and forest views

  • Path starts from: Happy Isles Trailhead
  • Path ends at: Nevada Fall
  • Distance: 8.2 miles round trip
  • Features: Gradual switchbacks, forest scenery
  • Difficulty: Moderate

Half Dome Trail

Location: Yosemite's most challenging day hike

Epic granite dome summit with cables

  • Path starts from: Happy Isles Trailhead
  • Path ends at: Half Dome summit (8,842 feet)
  • Distance: 16 miles round trip
  • Features: Cable route to summit
  • Permits: Required for cables (advance reservation)
  • Difficulty: Very strenuous

Mirror Lake Loop

Location: East end of Yosemite Valley

Easy walk to seasonal reflecting pool

  • Path starts from: Mirror Lake Trailhead
  • Path ends at: Mirror Lake
  • Distance: 2-5 miles (various routes)
  • Features: Seasonal lake, Half Dome reflections
  • Difficulty: Easy

Yosemite Falls Trail

Location: North side of Yosemite Valley

Steep trail to top of North America's tallest waterfall

  • Path starts from: Yosemite Falls Trailhead
  • Path ends at: Upper Yosemite Fall
  • Distance: 7.6 miles round trip
  • Elevation gain: 2,700 feet
  • Difficulty: Strenuous

Four Mile Trail

Location: Valley floor to Glacier Point

Historic trail connecting valley to high country

  • Path starts from: Valley floor trailhead
  • Path ends at: Glacier Point
  • Distance: 9.6 miles round trip
  • Elevation gain: 3,200 feet
  • Difficulty: Strenuous
